## Design Context

### Users
Windows users opening text files and very large logs who want a familiar Notepad-like experience instead of a custom utility shell. Their main job is to open files instantly, read them comfortably, and navigate them without the UI getting in the way.

### Brand Personality
Native, restrained, familiar. The interface should feel calm, trustworthy, and very close to Windows 11 Notepad rather than expressive or branded.

### Aesthetic Direction
Match the current Windows 11 Notepad look as closely as practical in WPF, with both light and dark themes. Use Windows-style chrome, simple menu structure, plain editor canvas, understated borders, and standard desktop spacing. Avoid decorative cards, bold accents, or custom dashboard styling.

### Design Principles
1. Prefer native-feeling Windows 11 patterns over custom styling.
2. Keep the editor surface plain and dominant; chrome should stay quiet.
3. Streaming performance is a UX feature, so visible content must appear immediately.
4. Light and dark themes should both feel first-party, not recolored afterthoughts.
5. When fidelity and novelty conflict, choose fidelity to Notepad.
